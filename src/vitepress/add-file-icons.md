# Add File Icons For Codeblocks in VitePress

## Note

By default there are no file icons in the VitePress.

I used `"vitepress-plugin-group-icons": "^1.6.5"` for File Icons.

Its so Easy and Simple.

::: warning
Install it as a `Dev Dependency`
:::
